## Overview

This skill provides guidance on applying the PERC design system, which is documented in `PERC-DESIGN-SYSTEM.md` in the project root.

## Usage

Trigger this skill when you need to ensure visual consistency across the project, using phrases like:

- "apply perc theme"
- "setup ui"
- "use design system"
- "make this look like PERC"

## Implementation Guide

Follow the guidelines in `PERC-DESIGN-SYSTEM.md` for:

1. **Theme Setup**: CSS variables and Tailwind configuration.
2. **Component Patterns**: Button, Card, Navbar, and Footer patterns.
3. **Responsive Guidelines**: Breakpoints and section padding rules.
4. **Quick Reference**: Tailwind class mappings for primary colors, shadows, and radii.

**Remember**: Always refer to `PERC-DESIGN-SYSTEM.md` for the most up-to-date component snippets.
